# منصة النتائج

أساس تطبيق عرض نتائج المسابقات والفعاليات المباشرة، مبني باستخدام React
وTypeScript وVite وTailwind CSS.

## التشغيل

```bash
npm install
npm run dev
```

## الأوامر

- `npm run dev` لتشغيل بيئة التطوير.
- `npm run build` للتحقق من الأنواع وإنشاء نسخة الإنتاج.
- `npm run lint` لفحص جودة الشفرة.
- `npm run preview` لمعاينة نسخة الإنتاج محليًا.

## البنية

- `src/components` للمكوّنات العامة ومكوّنات الواجهة القابلة لإعادة الاستخدام.
- `src/layouts` لتخطيطات التطبيق ومسار إعداد العرض.
- `src/pages` لصفحات التطبيق.
- `src/store` لحالة التطبيق المركزية.
- `src/services` لطبقة الوصول إلى البيانات والتخزين.
- `src/types` لأنواع نطاق التطبيق.
- `src/constants` للثوابت وخيارات العرض والمسارات.
- `src/hooks` للخطافات المشتركة.
- `src/utils` للأدوات المساعدة النقية.

تعتمد الحالة على واجهة `AppDataService`. التنفيذ الحالي يستخدم التخزين المحلي،
ويمكن تبديله لاحقًا بتنفيذ يعتمد على Supabase من نقطة التركيب في
`src/services/storage/app-data.service.ts`.
